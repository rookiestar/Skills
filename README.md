## Install

Clone into your OpenClaw skills directory:
```
cd ~/.openclaw/skills
git clone https://github.com/rookiestar/Skills.git
```

Then link or copy the skill folder you want, e.g.:
```
ln -s ~/.openclaw/skills/Skills/eng-lang-tutor ~/.openclaw/skills/eng-lang-tutor
```
